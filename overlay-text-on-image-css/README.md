# Add Text Over an Image with CSS

This folder contains the complete working code examples for adding text overlays to images using CSS.

## Folder Structure

```
overlay-text-on-image-css/
├── .env.example                           # Cloudinary credentials template
├── examples/                              # Working HTML examples
│   ├── 01-quick-start-step1.html         # Container setup
│   ├── 02-quick-start-step2.html         # Adding text
│   ├── 03-quick-start-step3.html         # Gradient background
│   ├── 04-positioning-corners.html       # Corner placements
│   ├── 05-positioning-center.html        # Center with transform
│   ├── 06-positioning-z-index.html       # Z-index layering
│   ├── 07-responsive-flexbox.html        # Flexbox centering
│   ├── 08-responsive-grid.html           # CSS Grid multi-position
│   ├── 09-responsive-aspect-ratio.html   # Aspect ratio containers
│   ├── 10-readability-text-shadow.html   # Text shadow techniques
│   ├── 11-readability-gradients.html     # Gradient overlays
│   ├── 12-readability-color-tints.html   # Color tint overlays
│   ├── 13-accessibility-semantic-html.html # Semantic HTML examples
│   └── cloudinary-text-overlay.ipynb     # Cloudinary Python examples
```

## How to View Examples

1. Open any HTML file directly in your browser
2. Or start a local server:
   ```bash
   cd examples
   python3 -m http.server 8080
   ```
   Then visit: http://localhost:8080/

## Cloudinary Setup

1. Copy `.env.example` to `.env`
2. Add your Cloudinary credentials
3. Open `examples/cloudinary-text-overlay.ipynb` in Jupyter
