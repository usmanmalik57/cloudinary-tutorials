Positioning: Z-Index Layering

Using z-index to control which elements appear on top when they overlap.

Blue Overlay Effect

Text appears above the color overlay

Layer Stack (bottom to top):

  • Layer 0: Image (default stacking)
  • Layer 1: Color Overlay (z-index: 1) — blue tint
  • Layer 2: Text Content (z-index: 2) — appears on top
