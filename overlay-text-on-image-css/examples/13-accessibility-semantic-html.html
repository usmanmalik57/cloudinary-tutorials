Accessibility Best Practices:

  • Always provide descriptive alt text that describes the image content
  • Use semantic HTML elements (<figure>, <figcaption>, headings)
  • Maintain minimum 4.5:1 contrast ratio for normal text (WCAG AA)
  • Don't rely on images alone to convey critical information
  • Keep important text in the DOM, not baked into images
