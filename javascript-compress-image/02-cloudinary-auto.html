Cloudinary Automatic Optimization

This page demonstrates how Cloudinary's URL parameters automatically optimize images

1. Quality Optimization (q_auto)

The q_auto parameter automatically selects the best compression quality

Without q_auto

URL:

/w_400/sample.jpg

With q_auto

URL:

/q_auto,w_400/sample.jpg

2. Format Optimization (f_auto)

The f_auto parameter automatically delivers WebP or AVIF to supporting browsers

JPEG (default)

URL:

/w_400/sample.jpg

With f_auto

URL:

/f_auto,w_400/sample.jpg

Your browser receives the best supported format automatically

3. Combined Optimization

Use both parameters together for maximum savings

No optimization

/w_400/sample.jpg

f_auto + q_auto

/f_auto,q_auto,w_400/sample.jpg

Typically 50-80% smaller
